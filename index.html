React Native Navigation

By Yasin Yusuf

What is Navigation in the context of react native

well, it means we can move between different screen in your app.

Navigation in the Web depends on URLs

You enter a url and you go to a page.

But in mobile there is no url that the user can type in and get to that page.

so its upto us developer to cover anything that the user might need.

Navigation will allow your users

To go forward

To go backward

and easy navigate around your app.

There are different patterns of navigation in the mobile.

They are called Navigators.

  • Stack
  • Drawer
  • Bottom Tabs
